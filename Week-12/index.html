Prom Dress Resale Website

My website is sort of like an online store, except only for used prom dresses.

  • When the user creates the account, they can upload a "profile" for their dress.
  • Once it sells, the buyer then becomes the owner of that dress profile and it becomes inactive.
  • Whenever that person is done with the dress, they can edit the profile (if necessary) and post it as active again.

Toolkits

I will use Bootstrap to create the website and make it easy to interact with. I want my website to look clean and polished.
